Guide to CherryOS

CherryOS is a free-to-install, lightweight operating system interface made by Sudeep.K

Installation and Setup:

  1. Navigate to the CherryOS directory using Terminal: cd path/to/cherryos_mac
  2. Launch CherryOS: open cherryos
  3. On first run, you'll be prompted to create a username and password
  4. CherryOS will automatically create shortcuts on your Desktop and in /usr/local/bin

Navigation:

  • Use 'W' and 'S' keys to move up and down in menus
  • Press Enter or Space to select an option

Main Features:

  • Applications: Quick access to your macOS Applications
  • File Manager: [Coming in future update]
  • Recovery Mode:
    • Full Reset - Resets user data (requires confirmation)
    • Shell - Access to system shell
  • Settings:
    • Display settings for header background and text
  • Shut Down: Exit CherryOS

Security:

CherryOS includes built-in security features with encrypted user credentials storage. Your password is required for each session.

Download:

You can download CherryOS by downloading the cherryos_mac folder in this repository.

After downloading, you may need to make the cherryos executable by running: chmod +x cherryos
