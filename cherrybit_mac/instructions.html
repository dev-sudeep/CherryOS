Guide to CherryBit

CherryBit is a free-to-install, lightweight operating system interface made by Sudeep.K

Installation and Setup:

  1. Navigate to the CherryBit directory using Terminal: cd path/to/cherrybit_mac
  2. Launch CherryBit: open cherrybit
  3. On first run, you'll be prompted to create a username and password
  4. CherryBit will automatically create shortcuts on your Desktop and in /usr/local/bin

Navigation:

  • Use 'W' and 'S' keys to move up and down in menus
  • Press Enter or Space to select an option
